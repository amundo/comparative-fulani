Comparative Fulani Lexicon

Why this is a good idea…

  • data on places like ucla phonetics archive, that australian langs website, etc. are great, but require a lot of clickthrough for comparison
    • so this gets data and displays it all at once, since the very structure is comparison
  • in some fieldwork there may be a lot of overlap in words elicited, such as work with Swadesh lists, work on multiple languages for parallel projects by the same/similar research teams, or work on related languages
    • so this automates comparisons between data with functionality to toggle the view of languages
  • audio files from elicitation sessions are often stored in long recordings or, if segmented, stored in individual folders and perhaps named inconsistently; navigating back and forth between files is clunky and doesn't allow for fast comparison and easy replay
    • audio files are associated with each datapoint (word/form) comparison is invited in the structure of the comparative lexicon and with user friendly play features
  • tool is useful for: a language learners, b researchers for their own work, c researchers for others' work
  • comparative meth
    • because of: a audio component, b comparison by gloss, c accessible gathered information
